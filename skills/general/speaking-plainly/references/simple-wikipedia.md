# Simple English Wikipedia Guidelines

Follow these core guidelines to make text accessible.

## Core Writing Rules
- **Short sentences**: Write simple sentences. Cover only one main idea per sentence.
- **Avoid compound sentences**: Limit conjunctions (like *and*, *or*, *but*, *however*, *therefore*).
- **Simple vocabulary**: Use common, basic English words.
- **Active voice**: Prefer active voice over passive constructions.
- **Simple tenses**: Stick to simple past or simple present tenses.
- **Do not shorten content**: Write simple language, not incomplete information. Explain difficult concepts in detail using simple terms.
- **Link complex terms**: Define or link to explanations of technical jargon immediately when they first appear.
